CKEditor اصلی

CKEditor یک برنامه منبع باز است که به این معنی است که می توانید هر گونه تغییری را که می خواهید تغییر دهید. این یک جامعه فعال است که دائما در حال توسعه برنامه با افزودنیهای رایگان و یک فرایند توسعه شفاف است.

فقط خواندنی CKEditor

API CKEditor امکان ویرایشگر را به حالت خواندن فقط می دهد. این حالت محتوای ویرایشگر را غیر قابل ویرایش می کند و کاربران نمی توانند آن را تغییر دهند.

رنگ رابط کاربری CKEditor

رنگ ویرایشگر UI را می توان توسط توسعه دهنده تنظیم کرد تا مطابق با ظاهر و احساس یک وب سایت یا یک برنامه باشد. برای تعریف رنگ رابط کاربر، از config.uiColor تنظیمات پیکربندی که کد رنگ RGB را می پذیرد. در مثال سردبیر زیر رنگ UI به تنظیم شده است #CCEAEE مقدار رنگ.

پیکربندی کلید را وارد کنید

این نمونه نشان می دهد چگونه پیکربندی Enter و Shift+Enter کلید برای انجام اقدامات مشخص شده در config.enterMode و config.shiftEnterModeتنظیمات، به ترتیب است.

محتوای چند زبانه

CKEditor مناسب برای محیط چند زبانه است. UI آن به زبان انگلیسی ترجمه شده است over 60 languages و به طور پیش فرض آن را در زبان کاربر نمایش داده می شود. علاوه بر این، کار با محتوای چند زبانه را بسیار راحت می کند با ارائه توانایی تنظیم جهت متن و علامت گذاری به زبان قطعات متن انتخاب شده.

ویرایش درون خطی

ویرایش درون خطی

در حال ویرایش درون خطی اجازه می دهد تا شما هر عنصر را در صفحه در محل ویرایش کنید. ویرایشگر درونی یک واقعیت را فراهم می کند ویسویگ تجربه "خارج از جعبه"، زیرا بر خلاف ویرایشگر کلاسیک, وجود ندارد <iframe> عنصر اطراف منطقه ویرایش. سبک CSS برای محتوای ویرایشگر مورد استفاده دقیقا همانند صفحه هدف است که این محتوا ارائه می شود!

ویرایش درونی توسط کد فعال شده است

دکمه "شروع ویرایش" را در زیر فشار دهید. یک ویرایشگر با استفاده از CKEDITOR.inline() روش توسط config.startupFocus تنظیم به true.
